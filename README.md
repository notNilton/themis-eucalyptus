# Gantt Chart Project  

Welcome to the **Gantt Chart Project**, a **TypeScript-based** interactive timeline visualization tool. This project provides an efficient way to **manage tasks, schedules, and project timelines** through an intuitive and customizable Gantt chart.

## Tech Stack  

This project is built with modern web technologies for flexibility and performance:  
- **TypeScript** – Ensures type safety and maintainability.  
- **React** – Enables a component-driven, dynamic UI.  
- **D3.js** – Used for rendering scalable and interactive charts.  
- **Tailwind CSS** – Provides a sleek, responsive design.  

## Features  

- **Dynamic Task Management** – Add, edit, and remove tasks in real-time.  
- **Drag & Drop Support** – Easily adjust task durations and dependencies.  
- **Customizable Themes** – Light and dark modes, adjustable colors, and styles.  
- **Zoom & Pan** – Navigate through project timelines with ease.  
- **Export & Import** – Save and load project data in JSON format.  

## Installation  

To set up and run the project locally:  

```sh
git clone https://github.com/yourusername/gantt-chart.git  
cd gantt-chart  
npm install  
npm run dev  
```

## Usage  

- Open the app in the browser and start adding tasks.  
- Drag tasks to adjust schedules.  
- Export your project timeline as JSON for future use.  

## Future Improvements  

- **Dependency Linking** – Connect tasks with dependencies for better project tracking.  
- **Multi-User Collaboration** – Enable real-time project updates with WebSockets.  
- **PDF/PNG Export** – Generate printable reports.
